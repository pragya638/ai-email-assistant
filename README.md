# 📧 Smart Email Assistant

AI-powered email reply generator built using Spring Boot, React, and Google Gemini API.

## 🚀 Live Demo

Frontend:
https://ai-email-assistant-frontend.onrender.com

Backend:
https://ai-email-assistant-mjac.onrender.com

## ✨ Features

- Generate professional email replies instantly
- Multiple tone options
  - Professional
  - Friendly
  - Formal
  - Casual
- Copy generated response with one click
- Responsive mobile-friendly UI
- REST API powered by Spring Boot
- AI-generated replies using Gemini API

## 🛠 Tech Stack

### Frontend
- React
- Vite
- Axios
- CSS3

### Backend
- Spring Boot
- Java
- REST API

### AI
- Google Gemini API

### Deployment
- Render

## ⚙️ Run Locally

### Backend

```bash
mvn spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Frontend:

```env
VITE_API_URL=http://localhost:8081
```

Backend:

```env
GEMINI_API_KEY=YOUR_API_KEY
```
